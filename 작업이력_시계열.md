- 구조: 정부 원본 파일 → Python 전처리 → 정적 JSON → React(Vite) → 단일 index.html
- 정리일: 2026-07-24

|---|---|---|

## 최근 수정 이력 (2026-07-24)

실데이터 연동 과정에서 두 번 어긋났고 아래처럼 복구했습니다.

| 문제 | 원인 | 조치 |
|---|---|---|
| 지역 경계가 네모 박스로 바뀜 | 소스 전달 zip에 진짜 경계가 아닌 테스트용 박스 파일이 섞임 | git 히스토리(`04f5e23`)의 실제 경계(1.27MB) 복원 |
| 격자가 인천 밖(부천·시흥)까지 퍼짐 | '인천 안' 판정 기준이 위의 박스 경계였음 | 실제 경계로 재필터 → 526셀 전부 인천 내부 |
| git 명령이 막힘 | `index.lock` 잔존 | 잠금 파일 제거 후 재실행 |
| "가운데가 빈다" 문의 | 오류 아님 | 바다·인천공항·미개발지 + SGIS 소규모 셀 비공개(합계의 약 76% 표출) |
| 군집분석(k-means)이 부적절 | 시군구 10개는 표본이 너무 작아 군집이 불안정, 옹진군이 "청년형"으로 오분류 | k-means 제거 → **1인가구율 × 고령1인비중, 인천 평균 기준 규칙기반 유형화**로 교체(산점도 ④와 동일 규칙, 투명·검증가능) |
